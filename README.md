# Java Practice

## 概要

Javaの学習を目的として作成したアプリケーションをまとめたリポジトリです。
基礎文法の理解から始め、複数のミニアプリを実装することで、プログラミングの実践力を身につけることを目指しました。

## 作成したアプリ一覧

* **メモアプリ** : 書き込み・表示・削除ができるアプリ
* **ToDoアプリ**：タスクの追加・表示・削除ができるアプリ
* **じゃんけんゲーム**：ユーザーとコンピュータで対戦するゲーム
* **サイコロアプリ**：ランダムな数値を生成するシンプルなプログラム
* **おみくじアプリ**：運勢をランダムで表示
* **RPG風ゲーム**：簡単な戦闘処理を実装したゲーム
* **金額計算アプリ**：税計算などを行うプログラム

## 使用技術

* Java
* ArrayList
* Scanner
* 制御構文（if文 / ループ処理）
* メソッド分割

## 工夫した点

* 各アプリで共通して「ユーザー入力」を扱うことで、実用的なプログラムを意識しました。
* エラーが発生しないように、入力値のチェック（条件分岐）を行いました。
* ToDoアプリでは、ユーザーが入力する番号と配列のインデックスの違いを考慮して実装しました。

## 苦労した点

* `nextInt()`と`nextLine()`の違いによる入力の不具合に苦労しました。
* 配列（ArrayList）のインデックス管理（0始まり）の理解に時間がかかりました。

## 今後の改善点

* ファイル保存機能の追加（データの永続化）
* GUIアプリへの発展
* コードの整理（クラス分割・設計の見直し）

## 実行方法

各Javaファイルをコンパイルして実行してください。

例：
javac ToDoApp.java
java ToDoApp
