# Java Practice

## 概要

Javaの基礎から応用までの理解を目的として、複数のミニアプリケーションを開発しました。
単なる文法学習にとどまらず、「実際に動くプログラム」を意識し、ユーザー入力・データ管理・ファイル操作などの実装を行いました。

---

## 作成したアプリ一覧

### ■ パスワード管理アプリ

* サイト名とパスワードの登録・一覧表示・削除が可能
* ファイル保存機能を実装し、データの永続化に対応
* 入力値のバリデーション（無効な番号のチェック）を実装

### ■ メモアプリ

* メモの登録・表示・削除が可能
* ファイル保存機能を実装

### ■ ToDoアプリ

* タスクの追加・一覧表示・削除
* インデックス管理を考慮した設計

### ■ じゃんけんゲーム

* ユーザー vs コンピュータの対戦
* ランダム処理を使用

### ■ サイコロアプリ

* 1〜6のランダムな数値を生成

### ■ おみくじアプリ

* 運勢をランダムに表示

### ■ RPG風ゲーム

* 簡単な戦闘ロジックを実装（HP管理・攻撃処理）

### ■ 金額計算アプリ

* 税込み計算などの数値処理を実装

---

## 使用技術

* Java
* ArrayList（動的配列）
* Scanner（ユーザー入力）
* BufferedWriter / FileWriter（ファイル出力）
* 制御構文（if文・for文・while文）
* メソッド分割による処理の整理

---

## 工夫した点

* ユーザー入力を前提とした設計を行い、実用的なアプリケーションを意識しました。
* 入力値のバリデーション（不正入力対策）を実装し、エラーの発生を防ぎました。
* 配列のインデックス（0始まり）とユーザー入力（1始まり）のズレを考慮して設計しました。
* ファイル保存機能を実装し、データの永続化を意識しました。
* パスワード管理アプリでは、削除処理において範囲チェックを行い、例外（IndexOutOfBoundsException）を防ぐ実装を行いました。

---

## 苦労した点・学んだこと

* `nextInt()`と`nextLine()`の違いによる入力バグに苦労し、Scannerの仕様理解を深めました。
* ArrayListのインデックス管理において、境界チェックの重要性を学びました。
* ファイル書き込み処理を通して、例外処理（try-catch）の必要性を理解しました。
* 条件分岐における論理演算子（&& / ||）の使い分けの重要性を学びました。

---

* ファイル読み込み機能の追加（アプリ起動時のデータ復元）
* クラス設計の見直し（データと処理の分離）

## 実行方法

以下のコマンドでコンパイル・実行できます。

```
javac ファイル名.java
java ファイル名
```

例：

```
javac PasswordApp.java
java PasswordApp
```
